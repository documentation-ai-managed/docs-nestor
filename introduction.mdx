## Overview

Nestor provides a centralized space for your project documentation. You create, organize, and publish docs effortlessly. Use it to build knowledge bases, API references, and user guides with version control and search.

<Callout kind="info">
Nestor integrates with your Git workflow. Changes to docs trigger automatic builds and deployments.
</Callout>

## Quick Start

Get started in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.nestor.app/register`.
  </Step>
  <Step title="New Workspace" icon="plus">
    Click "New Workspace" and name it "My Project Docs".
  </Step>
  <Step title="Upload Docs" icon="upload">
    Connect your Git repo or drag Markdown files.

````markdown
# Example doc.md
## Introduction
Welcome to your first doc.
````

  </Step>
  <Step title="Publish" icon="globe">
    Preview and publish. Share the live URL.
  </Step>
</Steps>

## Brand Guidelines

Use Nestor's brand color `#3B82F6` for consistency.

<Tabs>
  <Tab title="CSS Usage" icon="code">
    Apply the primary color in your custom themes.

````css
:root {
  --nestor-primary: #3B82F6;
  --nestor-primary-hover: #2563EB;
}

.button {
  background-color: var(--nestor-primary);
  color: white;
}
````

  </Tab>
  <Tab title="Guidelines" icon="file-text">
    - Primary: `#3B82F6` for buttons and links
    - Use on white or dark backgrounds
    - Avoid altering hue; adjust opacity for variants

    <Callout kind="tip">
      Test accessibility with WCAG tools.
    </Callout>
  </Tab>
</Tabs>

<Expandable title="Advanced Branding" default-open="false">
  Integrate SVG logos and custom fonts. Refer to `/configuration` for font setup.

````javascript
// Example theme config
const theme = {
  colors: {
    primary: '#3B82F6'
  },
  fonts: {
    body: 'Inter, sans-serif'
  }
};
````

</Expandable>
